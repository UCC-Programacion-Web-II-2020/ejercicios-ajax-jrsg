# Instrucciones

Los archivos de la práctica son:

1. index.html - la descripción de los ejercicios.
2. estilos.css - hoja de estilos.
3. ejercicios.js - para la solución de los 3 ejercicios, lado cliente.
4. paises.php - script para el ejercicio 1.
5. codigos.php - script para el ejercicio 2.
6. municipios.php - script para el ejercicio 3.
7. sql/codigos_postales.sql - script parar crear una tabla con los códigos postales de Veracruz.
8. sql/municipios.sql - script para crear una tabla con los municipios de México.
